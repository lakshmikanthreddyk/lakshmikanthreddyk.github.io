Deploying this website on GitHub is free, easy, and only takes a few steps:

  1. Create a free account on GitHub.com if you don't have one already.
  2. Create a new repository named lakshmikanth-reddy or my-portfolio. Make it Public.
  3. Inside your repository, create a new file named exactly index.html.
  4. Copy all the code from this document and paste it inside that file, then save (commit) the file.
  5. Go to the Settings tab of your repository. Scroll down on the left sidebar to find the Pages tab.
  6. Under "Build and deployment", select the main branch as the source and click Save.
